# 📊 Business Performance Dashboard (Excel)

## Overview

This project showcases an interactive Business Performance Dashboard built in Microsoft Excel. It provides insights into sales, profit, customer segments, product categories, regional performance, and the impact of discounts on profitability.

The dashboard was developed using Power Query, Pivot Tables, Pivot Charts, KPI Cards, Excel Formulas, and Slicers to support dynamic business analysis and reporting.

- Top 10 Performing States

## Workbook Structure

- *01_Data* – Cleaned Dataset
- *02_Pivot_Tables* – Pivot Tables
- *03_Dashboard* – Interactive Dashboard
- *04_Helper_Sheet* – Helper Formulas for KPI Calculations

---

## Notes

- Helper formulas are stored separately to maintain a clean dashboard layout.
- KPI cards update dynamically based on slicer selections.
- The dashboard enables interactive business analysis through dynamic filtering.
- A *Key Insights & Recommendations* section summarizes important business findings and suggested actions based on the dashboard analysis.
